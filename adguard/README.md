# AdGuard Home

В этой директории будут храниться заметки по установке и настройке AdGuard Home.

Планируемое содержимое:

- установка AdGuard Home в LXC-контейнер;
- настройка DNS-фильтрации;
- локальные DNS-записи;
- рекомендации по upstream DNS;
- примеры обезличенных конфигураций.

Реальный файл `AdGuardHome.yaml`, база данных и рабочая директория AdGuard не должны публиковаться в репозитории.
