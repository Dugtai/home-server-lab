# Proxmox

В этой директории будут храниться заметки и примеры конфигураций, связанные с Proxmox VE.

Планируемое содержимое:

- схема сетевых мостов `vmbr0` и `vmbr1`;
- план виртуальных машин и LXC-контейнеров;
- заметки по хранилищу;
- примеры обезличенных конфигураций;
- рекомендации по резервному копированию.

Реальные конфигурационные файлы, резервные копии ВМ и секреты не должны публиковаться в репозитории.
