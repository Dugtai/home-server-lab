# Diagrams

В этой директории будут храниться схемы проекта.

Планируемые схемы:

- общая топология сети;
- схема Proxmox bridge interfaces;
- схема маршрутизации трафика;
- схема DNS-фильтрации;
- схема резервного копирования.

Перед публикацией схем нужно удалять реальные публичные IP-адреса, MAC-адреса и другие чувствительные данные.
