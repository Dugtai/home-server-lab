# Screenshots

В этой директории будут храниться обезличенные скриншоты настройки и результатов работы проекта.

Перед публикацией скриншотов необходимо скрывать:

- публичные IP-адреса;
- MAC-адреса;
- серийные номера устройств;
- токены и ключи;
- логины;
- реальные домены и DDNS-имена;
- данные провайдера.
