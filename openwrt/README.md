# OpenWrt

В этой директории будут храниться заметки и примеры конфигураций OpenWrt.

Планируемое содержимое:

- настройка OpenWrt VM;
- WAN/LAN интерфейсы;
- DHCP и DNS;
- firewall notes;
- PBR / selective routing;
- VPN routing examples;
- anti-DPI routing notes for YouTube and Discord.

Реальные конфиги `/etc/config/network`, `/etc/config/firewall`, `/etc/config/dhcp`, VPN-ключи и резервные копии не должны публиковаться в репозитории.
