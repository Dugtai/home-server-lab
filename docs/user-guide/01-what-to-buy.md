# 01. What to Buy

Этот документ описывает минимальный и рекомендуемый набор оборудования для повторения проекта домашнего сервера-шлюза.

Цель: собрать компактный домашний сервер на базе мини-ПК, который будет выполнять роли маршрутизатора, DNS-фильтра, VPN-шлюза и платформы для дополнительных self-hosted сервисов.

## Минимальный комплект

| Компонент | Назначение | Статус |
|---|---|---|
| Mini PC с x86-64 CPU | Основной сервер под Proxmox VE | Обязательно |
| 16 GB RAM | Память для Proxmox, OpenWrt VM, LXC и сервисов | Желательно |
| SSD 256–512 GB | Системный диск и место под сервисы | Обязательно |
| 2 × Ethernet ports | Разделение WAN и LAN | Очень желательно |
| Wi-Fi access point | Раздача Wi-Fi клиентам | Обязательно, если нужен Wi-Fi |
| 2 × Ethernet cable | Подключение WAN и LAN/AP | Обязательно |
| USB flash drive 8–16 GB | Установка Proxmox VE | Обязательно |

## Почему нужен мини-ПК с двумя Ethernet-портами

Для роли домашнего шлюза желательно иметь два отдельных сетевых интерфейса:

```text
WAN port -> интернет от провайдера
LAN port -> домашняя сеть / точка доступа
```

Так проще построить понятную и безопасную схему:

```text
Provider router / ONT
        ↓
WAN port on Mini PC
        ↓
OpenWrt VM
        ↓
LAN port on Mini PC
        ↓
Access Point / switch
        ↓
Home devices
```

Если у мини-ПК только один Ethernet-порт, проект тоже можно реализовать, но потребуется управляемый коммутатор с VLAN. Для начинающего пользователя вариант с двумя Ethernet-портами проще и надёжнее.

## Почему 16 GB RAM достаточно

Для планируемой инфраструктуры 16 GB RAM достаточно на стартовом этапе:

| Сервис | Примерное потребление RAM |
|---|---|
| Proxmox VE | 1–2 GB |
| OpenWrt VM | 512 MB – 1 GB |
| AdGuard Home LXC | 256–512 MB |
| Chatbot LXC | 512 MB – 2 GB |
| Nextcloud test VM/LXC | 3–4 GB |
| Reserve | Остаток под кэш и будущие сервисы |

На первом этапе Nextcloud лучше не устанавливать. Сначала нужно стабилизировать сеть, DNS, VPN и маршрутизацию.

## Достаточно ли SSD на 512 GB

512 GB достаточно для:

- Proxmox VE;
- OpenWrt VM;
- AdGuard Home;
- чатбота;
- документации;
- тестового Nextcloud;
- небольших локальных бэкапов.

Но 512 GB недостаточно для полноценного домашнего облака с большим количеством фото, видео и резервных копий.

Для Nextcloud в будущем желательно добавить отдельный диск:

```text
System SSD 512 GB -> Proxmox, VM, LXC, system services
External/Internal storage -> Nextcloud data and backups
```

## Что желательно докупить позже

| Компонент | Зачем нужен |
|---|---|
| UPS | Чтобы сервер корректно переживал кратковременные отключения питания |

## Что проверить перед покупкой мини-ПК

Перед покупкой мини-ПК желательно проверить:

- поддерживает ли CPU виртуализацию Intel VT-x / AMD-V;
- есть ли минимум 2 Ethernet-порта;
- какой тип сетевых контроллеров установлен;
- можно ли включить автостарт после потери питания;
- можно ли загрузиться с USB;
- какой тип SSD используется;
- поддерживается ли замена SSD/RAM;
- есть ли нормальное охлаждение;
- есть ли отзывы по стабильности работы 24/7.

## Что проверить перед покупкой точки доступа

Перед покупкой access point желательно проверить:

- поддерживает ли устройство режим Access Point;
- можно ли отключить DHCP/NAT;
- какой стандарт Wi-Fi поддерживается;
- есть ли 2.4 GHz и 5 GHz;
- требуется ли PoE-питание;
- есть ли PoE-инжектор в комплекте;
- можно ли задать статический IP;
- есть ли веб-интерфейс управления.

## Рекомендуемый стартовый порядок покупки

1. Mini PC.
2. USB-флешка для установки Proxmox.
3. Ethernet-кабели.
4. Access Point.
5. Внешний диск для бэкапов.
6. UPS.
7. Managed switch, если позже понадобятся VLAN.

## Минимальная схема на старте

На первом этапе можно использовать роутер провайдера не в bridge mode, а как обычный роутер. Это упростит первичную настройку и снизит риск остаться без интернета.

```text
Provider router
        ↓
Mini PC with Proxmox and OpenWrt VM
        ↓
Access Point
        ↓
Home devices
```

После проверки стабильности можно перевести роутер провайдера в bridge mode и сделать OpenWrt основным маршрутизатором.

```text
Mini PC Intel N100 / 16 GB RAM / 512 GB SSD / 2 Ethernet ports

+
```
