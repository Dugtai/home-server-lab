# 02. First Start

Этот документ описывает первые действия после покупки мини-ПК и точки доступа.

Цель этапа: проверить оборудование, подготовить BIOS и создать установочную флешку с Proxmox VE.

## 1. Проверить комплект

Перед установкой нужно проверить, что есть всё необходимое:

- mini PC;
- блок питания mini PC;
- USB-флешка 8–16 GB;
- минимум два Ethernet-кабеля;
- access point;
- питание для access point или PoE-инжектор/PoE-коммутатор;
- монитор или телевизор с HDMI;
- клавиатура;
- другой рабочий компьютер для подготовки флешки.

## 2. Проверить mini PC перед установкой

Перед удалением Windows желательно один раз включить mini PC и проверить базовые параметры:

- устройство включается;
- SSD определяется;
- RAM определяется;
- оба Ethernet-порта физически работают;
- вентилятор и охлаждение работают нормально;
- нет циклических перезагрузок;
- можно зайти в BIOS/UEFI.

Если на mini PC установлена пробная Windows, сохранять её для этого проекта не нужно. Proxmox VE будет установлен на SSD полностью, Windows будет удалена.

## 3. Зайти в BIOS/UEFI

Обычно вход в BIOS выполняется одной из клавиш при включении устройства:

```text
Delete
F2
F7
F11
F12
Esc
```

Точная клавиша зависит от конкретной версии BIOS.

## 4. Настройки BIOS/UEFI

В BIOS желательно проверить и включить следующие параметры.

### Virtualization

```text
Intel Virtualization Technology / VT-x -> Enabled
```

Эта опция нужна для корректной работы виртуальных машин в Proxmox VE.

### Secure Boot

```text
Secure Boot -> Disabled
```

Secure Boot лучше отключить, чтобы не возникало проблем с установкой и загрузкой Proxmox VE.

### Boot from USB

```text
USB Boot -> Enabled
```

Эта настройка нужна для установки Proxmox VE с флешки.

### Restore on AC Power Loss

```text
Restore on AC Power Loss -> Power On
```

Эта настройка включает автоматический запуск mini PC после восстановления питания.

Для домашнего сервера это важно: если отключат электричество, сервер сможет включиться сам.

### Wake-on-LAN

```text
Wake-on-LAN -> Enabled
```

Опционально. Может пригодиться для удалённого включения устройства по сети.

## 5. Подготовить установочную флешку

На рабочем компьютере нужно скачать:

- ISO-образ Proxmox VE;
- программу для записи образа на USB-флешку.

Подходящие программы:

- Rufus;
- balenaEtcher;
- Ventoy.

Для начинающего пользователя проще использовать Rufus или balenaEtcher.

## 6. Записать Proxmox VE на флешку

Общий порядок:

1. Вставить USB-флешку в рабочий компьютер.
2. Открыть Rufus или balenaEtcher.
3. Выбрать ISO-образ Proxmox VE.
4. Выбрать USB-флешку.
5. Запустить запись.
6. Дождаться завершения.
7. Безопасно извлечь флешку.

Важно: все данные на флешке будут удалены.

## 7. Подключить mini PC для установки

Для установки потребуется:

```text
Mini PC -> HDMI -> Monitor/TV
Mini PC -> USB -> Keyboard
Mini PC -> USB -> Proxmox installer flash drive
Mini PC -> Ethernet -> existing router/network
```

На первом этапе лучше подключить mini PC к существующему роутеру провайдера или домашнему роутеру. Не нужно сразу переводить роутер провайдера в bridge mode.

## 8. Почему не нужно сразу включать bridge mode у провайдера

На первом этапе безопаснее использовать временную схему:

```text
Provider router
        ↓
Mini PC
```

Так даже при ошибках в настройке Proxmox или OpenWrt основная домашняя сеть не будет полностью сломана.

Bridge mode лучше включать только после того, как:

- Proxmox установлен;
- OpenWrt VM настроена;
- LAN работает;
- клиенты получают IP;
- интернет через OpenWrt стабилен;
- доступ к Proxmox не потерян.

## 9. Что записать перед установкой

Перед установкой полезно записать:

```text
Model: Firebat S1
CPU: Intel N100
RAM: 16 GB
SSD: 512 GB
Ethernet port 1: unknown before Proxmox
Ethernet port 2: unknown before Proxmox
Temporary network: existing router DHCP
```

Названия сетевых интерфейсов будут понятны уже после установки Proxmox.

Они могут выглядеть примерно так:

```text
enp1s0
enp2s0
```

## 10. Контрольный список перед установкой Proxmox

Перед переходом к установке должно быть выполнено:

- mini PC включается;
- BIOS доступен;
- VT-x включён;
- Secure Boot отключён;
- USB Boot включён;
- Proxmox VE записан на флешку;
- есть клавиатура и монитор;
- mini PC подключен к сети Ethernet;
- важные данные с Windows не нужны или уже сохранены.
