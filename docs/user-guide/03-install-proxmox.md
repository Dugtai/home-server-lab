# 03. Install Proxmox VE

Этот документ описывает базовую установку Proxmox VE на mini PC.

Цель этапа: установить Proxmox VE на SSD mini PC и получить доступ к веб-интерфейсу управления.

## 1. Важное предупреждение

Установка Proxmox VE на SSD полностью удалит установленную Windows и все данные на диске.

Перед установкой нужно убедиться, что:

- важные файлы сохранены;
- Windows на mini PC больше не нужна;
- установочная флешка с Proxmox VE подготовлена;
- есть монитор и клавиатура.

## 2. Загрузиться с USB-флешки

1. Вставить флешку с Proxmox VE в mini PC.
2. Подключить монитор и клавиатуру.
3. Включить mini PC.
4. Открыть Boot Menu.
5. Выбрать USB-флешку.

Обычно Boot Menu открывается клавишами:

```text
F7
F11
F12
Esc
```

Точная клавиша зависит от BIOS.

## 3. Запустить установку

В меню Proxmox VE выбрать:

```text
Install Proxmox VE (Graphical)
```

Если графический установщик не запускается, можно попробовать:

```text
Install Proxmox VE (Terminal UI)
```

## 4. Принять лицензионное соглашение

На первом экране установщика нужно принять лицензионное соглашение.

## 5. Выбрать диск установки

Выбрать основной SSD mini PC.

Для текущего проекта:

```text
SSD: 512 GB
Target disk: internal SSD
```

Файловую систему на старте можно оставить стандартную, которую предлагает установщик.

Для простого домашнего стенда не нужно усложнять установку ZFS, если в mini PC установлен только один SSD. ZFS имеет смысл рассматривать позже, если будет несколько дисков и понятная стратегия резервного копирования.

## 6. Настроить регион и клавиатуру

Пример:

```text
Country: Russia / Latvia / другое фактическое местоположение
Time zone: выбрать свой часовой пояс
Keyboard layout: English или Russian/English
```

Для серверов обычно удобнее оставить английскую раскладку по умолчанию.

## 7. Задать пароль и email

Установщик попросит задать пароль для пользователя:

```text
root
```

Требования:

- пароль должен быть сложным;
- пароль нужно сохранить в надёжном месте;
- не использовать простые пароли вида `admin`, `password`, `12345678`.

Email можно указать свой рабочий или личный.

## 8. Настроить временную сеть Proxmox

На этапе установки Proxmox нужно выбрать сетевой интерфейс и задать IP.

Если mini PC подключён к существующему роутеру, можно использовать временную адресацию из текущей домашней сети.

Пример:

```text
Hostname: pve.home.arpa
IP address: 192.168.1.10/24
Gateway: 192.168.1.1
DNS server: 192.168.1.1
```

Это временная настройка. Позже, после создания OpenWrt VM, Proxmox будет перенесён в новую LAN-сеть, например:

```text
Proxmox: 192.168.10.2/24
Gateway: 192.168.10.1
DNS: 192.168.10.3 или 192.168.10.1
```

## 9. Завершить установку

После проверки настроек нажать Install и дождаться завершения установки.

После завершения:

1. Извлечь USB-флешку.
2. Перезагрузить mini PC.
3. Дождаться загрузки Proxmox VE.

## 10. Войти в веб-интерфейс

На другом компьютере в той же сети открыть браузер:

```text
https://192.168.1.10:8006
```

Адрес зависит от IP, который был указан при установке.

Вход:

```text
User name: root
Password: пароль, заданный при установке
Realm: Linux PAM standard authentication
```

Браузер может показать предупреждение о небезопасном сертификате. Для локального сервера на этом этапе это нормально, потому что Proxmox использует самоподписанный сертификат.

## 11. Первичная проверка после установки

После входа в Proxmox проверить:

- открывается веб-интерфейс;
- виден node Proxmox;
- виден локальный SSD;
- есть сетевой интерфейс;
- сервер не перезагружается самопроизвольно;
- дата и время корректные.

## 12. Что не делать сразу

Не нужно сразу:

- переводить роутер провайдера в bridge mode;
- открывать Proxmox в интернет;
- создавать все ВМ одновременно;
- устанавливать Nextcloud;
- настраивать VPN;
- менять много сетевых параметров без записи текущего состояния.

Сначала нужно стабилизировать базовую установку.
