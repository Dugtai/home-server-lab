# User Guide

Этот раздел предназначен для пошагового повторения проекта с нуля.

Целевая аудитория: пользователь без глубокого опыта в Linux, Proxmox, OpenWrt и сетевой инфраструктуре.

Задача раздела — объяснить настройку максимально практично:

- что купить;
- куда подключить кабели;
- что скачать;
- какие параметры выбрать;
- как проверить результат;
- что делать при типовых ошибках.

Документация в этом разделе должна быть написана простым языком и идти строго по шагам.
