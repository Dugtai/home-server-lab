# Services

В этой директории будут храниться материалы по дополнительным сервисам домашнего сервера.

Планируемые сервисы:

- chatbot;
- Nextcloud;
- monitoring;
- backup tools.

Для каждого сервиса должны использоваться только обезличенные примеры конфигураций. Токены, пароли, ключи и реальные `.env` файлы не должны публиковаться.
